The Grid Reaper

It is a magic CSS file.

You can see the magic at work in the Demo.

With minimal interruption to your HTML content, and using highly sophisticated naming conventions, The Grid Reaper should help you quickly and efficiently create a scaffolding structure to any HTML / CSS based web project.
